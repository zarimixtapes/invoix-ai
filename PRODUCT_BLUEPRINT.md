# BuildMind AI product blueprint

## The promise
One voice update from site becomes every construction record needed: diary, delay evidence, variation draft, defect/hazard/task, client update, HQ update, architect query and searchable project memory.

## Crucial feature set included in this codebase
1. Voice-to-everything AI endpoint
2. Site diaries
3. Delay register
4. Variation register
5. Hazards
6. Defects
7. Timesheets
8. Project Memory search
9. Draft emails to client, HQ, architect, contractor, subcontractor
10. Google Drive sync export hook
11. Team roles and permissions
12. Client portal-ready role separation
13. 14-day trial
14. Stripe subscription checkout and webhook
15. Trial guardrails
16. Audit logs
17. Dashboard UI and feature pages

## What makes it different from generic construction software
- Voice-first: less typing and forms.
- Revenue protection: variations and delays captured early.
- Dispute defence: everything becomes a timeline.
- Role-based access: workers log basics; clients see polished updates; managers see risks and money.
