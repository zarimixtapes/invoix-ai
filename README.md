# Invoix AI Full Web App

A much fuller MVP:
- dashboard
- light/dark mode
- invoice builder
- AI + browser voice invoice creator
- editable templates
- customers
- PDF/print
- Stripe checkout route

Upload all files to GitHub, commit, then Vercel will redeploy.

Environment variables:
OPENAI_API_KEY
STRIPE_SECRET_KEY
NEXT_PUBLIC_STRIPE_PRO_MONTHLY_PRICE_ID
NEXT_PUBLIC_SITE_URL

This is a strong MVP, not a full Xero clone yet. Xero-level features require auth, database per user, accounting reports, tax rules, email sending and webhooks.
